# JS 30

![Banner](https://github.com/destiny5420/JS-30/blob/develop/page_source/banner.png)
JS30 是一個 Javascript 的線上教學課程，利用三十天的時間，每天實作簡單的 JS 作品，並以初學者的角度講解相關的概念，也讓我透過實作的練習，更加熟悉 Javscript 的語法特性與功能，並在此做紀錄與效果演示。
